interactive map of Eindhoven

a dynamic map project for Stichting 18 September.

visit website

The Interactive Map of Eindhoven is a web project designed entirely by hand, without relying on pre-existing frameworks, using HTML, CSS, and JavaScript. As the main developer, I focused on creating a dynamic map with seamless zoom, drag, and interactive marker functionalities, providing users with a smooth and engaging experience.

Key features include a dynamic zoom function that ensures optimal visibility on any device, draggable navigation for intuitive exploration, and interactive pop-ups offering historical insights and Google Maps directions. The decision to hand-code the entire project allowed for full customization and a deeper understanding of the technical challenges.

The development process included resolving conflicts between different zoom implementations and integrating custom pop-up designs while ensuring compatibility. Overcoming these challenges significantly improved my coding and problem-solving skills.
