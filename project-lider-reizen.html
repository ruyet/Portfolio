visit website

Lider Reizen is a branding and UI/UX project focusing on delivering a seamless experience for travelers. This project included designing a complete brand identity, a fully functional Landing Page, and ensuring a consistent user experience across platforms.

The website is built with a focus on modern design principles and performance. I have used HTML and CSS for this project. My initial goal was to re-create the design I made for this Landing Page using Figma, in code. This was a great opportunity for me to learn how to initially re-create a design in code.
